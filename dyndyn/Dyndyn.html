Module Dyndyn


module Dyndyn: sig  end
This module implements arrays that are extensible from both sides.
type 'a t
The abstract type of double-extensible arrays. They can be considered as mutable double-ended lists as well as regular arrays. The first index is 0 as in the standard array type.
val create : unit -> 'a t
creation of an empty array
val compact : 'a t -> unit
resize the array to minimize its memory occupancy
val add_first : 'a t -> 'a -> unit
add an element at the beginning of the list. Index of element i becomes i+1.

add an element at the end of the list
