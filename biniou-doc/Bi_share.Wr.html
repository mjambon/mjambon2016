val put : tbl -> 'a * Bi_share.type_id -> int -> int
put tbl x pos returns 0 if x is not already in the table and adds x to the table. pos is the absolute position of the first byte of the ref value excluding its tag. If x is found in the table, then the difference between pos and the original position is returned.
