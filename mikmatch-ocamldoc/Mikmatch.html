module Mikmatch: sig .. end
A small text-oriented library

The Mikmatch module provides a submodule named Text. A normal usage is to place open Mikmatch at the beginning of user code that uses it.

This module is part of the runtime environment of Mikmatch (the library run_mikmatch_pcre.cma or equivalent).
